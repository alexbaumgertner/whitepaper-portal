# pt-list

![pt-list](_images/pt-list.png)

`pt-list` – это описание списка похожих друг на друга вертикально перечисленных сущностей.

Структура блока очень простая:

```json
{
	block: 'pt-list',
	content: [
	{
		elem: 'item'
	},
	{
		elem: 'item'
	}]
}
```
